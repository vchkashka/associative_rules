# Изучение ассоциативных правил

## Данные

[Online Retail Data Set](https://www.kaggle.com/datasets/mathchi/online-retail-data-set-from-ml-repository) - это набор данных о транзакциях британского онлайн-ритейлера, специализирующегося на уникальных подарках для всех возрастов, за период с 01.12.2010 по 09.12.2011. Данные представлены в файле `retail_dataset.csv`.

## Задача

Исследование методов анализа ассоциативных правил и сравнение алгоритмов Apriori и FP-Growth на данных о покупках.

## Используемые библиотеки
*pandas*   
*mlxtend*  
*seaborn*   
*networkx*   
*matplotlib*
